## Run Locally

Clone the project

```bash
  git clone https://github.com/Redstoneguy129/Ani4Plex.git
```

Go to the project directory

```bash
  cd Ani4Plex
```

Install dependencies

```bash
  npm install
```

Start Ani4Plex

```bash
  npm run start
```


## Features

- Light/dark mode toggle
- Auto season mapping
- Web user interface
- Create connections (Discord, Anilist, MAL)
- Authentication
- Cross platform
